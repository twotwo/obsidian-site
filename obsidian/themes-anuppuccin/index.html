AnuPpuccin(/‘ænjuːputʃin/) 是一个为 Obsidian 设计的个人主题，由 AnubisNekhet 开发并在 GitHub 上开源。该主题支持多种颜色方案，包括 Catppuccin、Dracula 等，并提供了两种布局选项。用户可以通过 Git 命令行安装和更新主题。

Features

  • 自定义调色板 Custom palette
  • 两种布局变体 2 layout variants
  • 彩虹文件夹
  • 自定义边框 Frames
  • 自定义 Decorations，斜体和粗体的文本
  • 自定义 Callout
  • 彩色 Checkbox
  • 标记样式 CSS
  • 推荐字体：界面 Overpass，文本 Rubik，标题 Noto Serif

主题的默认主题是 Latte 和 Mocha，分别用于浅色模式和深色模式。

主题支持 Excalidraw、Kanban 和 Style Settings 插件的集成。

项目状态

因开发者的学业和项目任务，项目处于休眠状态，暂时可能无法及时修复 bug 和问题。

目前版本 v1.5.0, 2024/5/11
