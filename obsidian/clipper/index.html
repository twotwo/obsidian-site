Obsidian Web Clipper

Obsidian 官方页面剪藏插件

2024-11-07
1 minute read

Obsidian Web Clipper1 是一个可以在不同浏览器上安装的插件，用于提取网页内容并将其保存到 Obsidian 笔记库。

Obsidian Web Clipper 支持剪辑网页的文本、图片和链接，并以 Markdown 文件的形式存储在 Obsidian 笔记库中。用户可以选择性地去除广告并仅保存特定内容，从而获得更清晰的剪辑效果。保存后，内容可以在 Obsidian 中编辑、组织和链接，实现高效的研究流程。

1. 安装

Obsidian Web Clipper 支持 Chrome、Firefox、Safari 等浏览器，需要安装 Obsidian 1.7.2 或更高版本1。

iOS/ iPadOS 可以从应用商店直接下载：

2. 保存页面

使用时，可以通过浏览器工具栏图标、热键或上下文菜单激活插件，然后选择 “Add to Obsidian” 按钮来保存网页。

插件界面分为四个部分：

  1. 标题（可以切换模板、启用高亮模式和访问设置）
  2. 属性（显示从网页中提取的元数据）
  3. 笔记内容（将被保存到 Obsidian）
  4. 页脚（可以选择保存到哪个库和文件夹，并提供保存到 Obsidian 的功能）

3. 高亮网页文本

Web Clipper 还支持高亮网页文本的功能，允许用户选择要保存到 Obsidian 的文本、图片和元素。用户可以通过插件面板中的高亮图标、热键或上下文菜单激活高亮模式。此外，用户可以在设置中更改高亮行为，并将高亮内容导出为 .json 文件。

4. 模版

用户还可以创建和编辑模板，以自动捕获和组织来自网页的元数据。模板可以通过新建、编辑、导入和导出进行管理。模板触发器允许根据当前页面的 URL 或 schema.org 数据自动选择模板。模板变量可以用于自动填充笔记名称、位置、属性和内容，支持预设变量、元数据变量、选择器变量和 schema.org 变量。2

5. 排障

如果用户在使用 Web Clipper 时遇到问题，可以通过官方 Discord 频道寻求帮助，或者在 GitHub 仓库 中报告错误。

对于在 Linux 系统上可能遇到的问题，提供了一些故障排除步骤。
