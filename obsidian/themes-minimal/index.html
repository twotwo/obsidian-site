Obsidian Theme: Minimal

2024-08-10
2 minute read

Minimal 是一个为 Obsidian 知识库设计的主题，由 Steph Ango 维护，旨在提供无干扰的写作和阅读体验。主题遵循 MIT 许可证，并且在 GitHub 上有超过 3.6k 的星标。

外观设置(v1.6.7)

基础参数

  1. Appearance -> Base color scheme(Dark/Light)，确定基础颜色；

  2. Appearance -> Accent color，设置主题色。

    Favorite Color

  3. Appearance -> Themes

    安装主题：Minimal1 v7.7.14

    安装伴侣插件： Minimal Theme Settings / Style Settings

Minimal Document

https://minimal.guide/

Minimal 主题支持多种背景风格，包括低对比度、高对比度以及适合 OLED 显示屏的真黑模式。用户可以通过 Minimal Theme Settings 和 Hider 插件来自定义颜色、字体和其他界面元素。此外，Minimal 支持多种流行插件，如 Calendar、Kanban、Dataview 等，并提供了多种辅助类和图片滤镜来增强文档的布局和样式。用户还可以通过 Style Settings 插件创建自定义颜色方案，并且可以使用快捷键来控制布局和样式。

App-level features

  • 活动行高亮：提高编辑时的可视性 Style Settings -> Line Numbers
  • 背景对比度：多种背景对比度的选项 Minimal Theme Settings
  • 颜色方案：选择和定制颜色方案 Minimal Theme Settings
  • 彩色边框(0.16+ 支持新的 Tabs 功能)：Minimal Theme Settings
  • 电子墨水显示模式：对单色显示的支持 Minimal Theme Settings
  • 专注模式：Minimal Theme Settings
  • 快捷键：Cmd + P -> Toggle focus mode
  • 图片缩放：Style Settings, Minimal › Images › Disable image zoom
  • 行宽：Minimal Theme Settings 定义几种行宽和对应的块
  • 发布支持
  • 标签页样式：Style Settings -> Sidebars -> Sidebar tab style
  • 导航标签：将左上导航栏从默认的水平排列变为垂直排列 Minimal Theme Settings

Block-level features

  • Colorful Headings
  • Cards
  • Charts
  • Checklists
  • Embeds and transclusions

Minimal 主题如何提供深度自定义功能？

  1. Minimal Theme Settings 插件：允许用户自定义颜色方案、字体、热键，并访问主要的功能开关。用户可以选择不同的背景对比度，包括适合 OLED 屏幕的真黑模式，以及多种预设的颜色方案，如 Catppuccin、Dracula 等。

  2. Style Settings 插件：用于进一步自定义颜色、字体大小、字体样式等细节。用户可以在新的 Pane 中打开 Style Settings，以便更容易地看到自己的更改。

  3. 辅助类和图片滤镜：提供了多种 CSS 辅助类，用于控制图片、表格和 iframe 的宽度，以及将 Dataview 表格转换为卡片布局。此外，还有图片滤镜，如 invert、circle 和 outline，用于创建特定的图像效果。

  4. 快捷键：通过 Minimal Theme Settings 插件提供了多种快捷键，用于切换背景模式、布局宽度，以及启用或禁用焦点模式等功能，提高了用户操作的效率。

  5. 自定义颜色方案：用户可以使用 Style Settings 插件创建自己的颜色方案，并且可以为浅色模式和深色模式分别选择不同的方案。
